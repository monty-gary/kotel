# Rainbow Swirl (GitHub Pages)

Production-ready single-page site with:
- Animated rainbow full-page gradient background
- Brightness and Sharpness +/- controls
- Swirl particle effects reacting to mouse/pointer, touch, click/tap, and scroll
- Accessibility + performance fallbacks (reduced motion and no-canvas)

## Local Run

Open `index.html` directly in a browser, or serve with any static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy (GitHub Pages)

1. Push `main` to GitHub.
2. Enable GitHub Pages with source `main` and folder `/`.
3. Wait for Pages build to finish.

Live URL placeholder: `https://<owner>.github.io/<repo>/`
